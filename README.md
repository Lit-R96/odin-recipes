# odin-recipes

Note: English is not my main language

The Current project objective is to show what we learned with the Odin project until this point, using the git basic and html foundations to create a website
of recipes that will consist of a main index page which will have links to a few recipes.

At the end of the project, I will show the skills required to create a website from zero, only using html, from using the correct html boilerplate to adding text,
images, links in a proper way, demonstrating knowledge in basic elements and tags. To finally commit the project to github for future reviews.
